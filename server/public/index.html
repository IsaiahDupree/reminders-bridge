Install the Mac agent

A tiny helper runs on your Mac and answers your ChatGPT requests. Your notes stay on your machine — only the specific results you ask for are relayed.

1. Pair your Mac to this account (get your code in Step 2, then run):

npx apple-notes-agent pair <CODE>

2. Keep it running automatically after every restart:

npx apple-notes-agent install

Requires Node.js. On first run macOS will ask permission for the agent to control Apple Notes — click Allow.

2

Pair your Mac

Generate a one-time code, then run the pair command from Step 1 on your Mac.

This code expires in 10 minutes. It's already filled into the pair command in Step 1 — copy it and run it on your Mac.

3

Connect ChatGPT

Add NotesBridge as a custom connector in ChatGPT (one time).

  1. In ChatGPT, open Settings → Plugins (formerly Connectors).
  2. Under Security & login, enable Developer mode.
  3. Choose Create a custom plugin (custom connector).
  4. Paste this MCP Server URL:
https://notesbridge.vercel.app/mcp
  1. Set Authentication to OAuth.
  2. ChatGPT sends you here to Sign in and click Allow — then you're connected.
4

Try it in ChatGPT

Start a new chat, enable the NotesBridge connector, and ask:

Search my Apple Notes for quarterly goals
Create a note titled Meeting notes with today's action items

Your Mac must be awake with the agent running for requests to work. Write actions are always confirmed inside ChatGPT before they run.
